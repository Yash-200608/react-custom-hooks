# React Custom Hooks (useFetch)

## Overview
This project demonstrates the use of a custom React hook named `useFetch`
to fetch data from an external API while handling loading and error states.

## Features
- Custom useFetch hook
- Handles loading and error states
- Reusable data fetching logic
- Built using React and Vite
